🛒 E-commerce Analytics – Olist Dataset
📌 Project Overview

This project performs end-to-end data analysis on the Olist E-commerce dataset to extract meaningful business insights. It explores sales performance, customer behavior, delivery efficiency, product trends, and seller performance using data analytics techniques.

The goal is to transform raw transactional data into actionable insights that support better decision-making in e-commerce business operations.

📊 Objectives
Analyze overall sales and revenue trends
Understand customer purchasing behavior
Evaluate delivery performance and delays
Identify top-selling product categories
Study seller performance and ratings
Measure customer satisfaction levels
🧰 Tools & Technologies
Python 🐍
Pandas & NumPy
SQL
Matplotlib & Seaborn
Jupyter Notebook
Power BI / Tableau (optional for dashboards)
📁 Dataset

The dataset used is the Olist Brazilian E-commerce Public Dataset, which includes:

Orders data
Customers data
Products data
Sellers data
Payments data
Reviews data
🔍 Key Insights
Revenue trends over time
High-demand product categories
Delivery time impact on customer reviews
Top-performing sellers
Customer segmentation based on order behavior
📈 Visualizations

The project includes:

Sales trend charts
Category-wise revenue analysis
Delivery time distribution
Customer review analysis
Geographic sales distribution
